## How to Deploy This ZIP

1. **Extract** the ZIP to a folder
2. **Open terminal** in that folder
3. **Install dependencies**: `npm install`
4. **Test locally**: `npm run dev`
5. **Push to GitHub**: 
   ```
   git init
   git add .
   git commit -m "Complete rebuild"
   git remote add origin https://github.com/devil8381-cyber/claimguardtort.git
   git push -f origin main
   ```
6. **Railway auto-deploys** from GitHub — just make sure env vars are set
